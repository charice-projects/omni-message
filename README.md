# Omni-Message - Android全渠道消息应用

## 📱 项目简介
一个聚合多种消息渠道的Android应用，实现一站式消息管理。

## ✨ 功能特性
- ✅ 多平台消息聚合（微信、QQ、短信、邮件等）
- ✅ 智能消息分类与过滤
- ✅ 统一消息提醒
- ✅ 快捷回复模板
- ✅ 消息统计分析

## 🚀 快速开始
### 环境要求
- Android Studio Arctic Fox 或更高版本
- JDK 11+
- Android SDK 30+
